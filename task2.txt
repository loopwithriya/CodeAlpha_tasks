# ✅ To-Do List App with GUI – CodeAlpha Task 2

This is a simple yet functional **To-Do List App** created using **Python's Tkinter GUI library**. It allows users to add, delete, save, and manage their tasks with ease.



## 📌 Features

- Add tasks with a single click
- Delete selected tasks
- Clear all tasks with confirmation
- Save tasks to a `.txt` file
- Automatically load saved tasks on startup
- Attractive and clean user interface using `Tkinter`
- Listbox with task highlighting
- Emoji-enhanced buttons for a fun user experience 🌟



## 🛠️ Technologies Used

- Python 3.x
- Tkinter (Standard GUI library)



## 🚀 How to Run

1. **Clone or Download** this repository  
2. Open the folder in **VS Code** or any Python IDE  
3. Run the Python file:

```bash
python todo_app.py
